Implemented by Nathan Rossol - Computer Science Undergraduate at the University of Michigan

Description: Interactive path-finding algorithm visualizer implemented in Python with the Pygame library.
	     Allows the user to choose custom start/end nodes, and design their own map by creating walls
	     that the algorithm will have to dodge. The user can choose among several algorithms to run.

Directions:
- Install the latest version of Python.
- Install the latest version of Pygame.
- Go to the directory where the downloaded code is located via your terminal, and run: "python3 pfv.py"
